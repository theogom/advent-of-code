# Advent of code 2022

The project is in Typescript.

The puzzles are in the `src/puzzles` folder.

The inputs are in the `inputs` folder.

## Commands

Build:
    
```bash
yarn build
```

Run all puzzles:
    
```bash
yarn start
```

Run a specific puzzle:
    
```bash
yarn start <puzzle-index>
```

Develop with watch:
    
```bash
yarn dev
```
