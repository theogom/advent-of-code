# Advent of code 2022

Days for the [advent of code 2022](https://adventofcode.com/2022).

The project is in Typescript.

The days are in the `src/days` folder.

The inputs are in the `inputs` folder.

## Commands

Build:

```bash
npm run build
```

Run all days:

```bash
npm run start
```

Run a specific day:

```bash
npm run start <day-index>
```

Develop with watch:

```bash
npm run dev
```
