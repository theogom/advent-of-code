# Advent of code 2015

Solutions for the [advent of code 2015](https://adventofcode.com/2015).

The project is in Python.

Solutions of each days are in the `src/days` folder.

The inputs are in the `inputs` folder.

## Commands

Run all days:
    
```bash
make
```

Run a specific day:
    
```bash
make DAY=<day_number>
```
